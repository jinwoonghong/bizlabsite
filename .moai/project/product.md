# bizlabsite - 연구실 논문 관리 서비스

## 프로젝트 개요

**bizlabsite**는 연구실 구성원들이 발표한 논문의 메타데이터와 원문 링크를 등록, 수정, 관리할 수 있는 반응형 웹 애플리케이션입니다. 연구실 내부 구성원뿐만 아니라 외부 방문자도 논문 목록을 열람하고 검색할 수 있도록 설계되었습니다.

---

## 대상 사용자

### 주요 사용자

- **연구실 구성원**: 자신의 논문을 등록하고 메타데이터를 관리하는 연구원, 대학원생, 교수
- **연구실 방문자**: 연구실의 연구 성과를 열람하고 관심 논문을 검색하는 외부 방문자, 협력 연구자

### 사용자 특성

- 별도의 회원가입 시스템 없이 누구나 논문을 등록할 수 있음
- 논문 수정 및 삭제는 관리자 비밀번호 입력이 필요함
- PC와 모바일 환경 모두에서 접근 가능

---

## 핵심 기능

### 1. 논문 CRUD (등록/조회/수정/삭제)

논문의 전체 생명주기를 관리하는 핵심 기능입니다.

- **등록**: 제목, 저자, 발행연도, 학술지/학회명, 원문 링크, 초록 등 메타데이터를 입력하여 논문을 등록
- **조회**: 등록된 논문 목록을 확인하고 개별 논문의 상세 정보를 열람
- **수정**: 관리자 비밀번호 인증 후 기존 논문의 메타데이터를 수정
- **삭제**: 관리자 비밀번호 인증 후 논문을 삭제

### 2. 관리자 비밀번호 보호

간단하면서도 효과적인 접근 제어 방식입니다.

- 논문 등록은 누구나 가능 (별도 인증 불필요)
- 논문 수정/삭제 시에만 관리자 비밀번호 입력 필요
- 별도의 사용자 등록/로그인 시스템 없음
- 관리자 비밀번호는 환경 변수로 관리

### 3. 검색 및 필터링

원하는 논문을 빠르게 찾을 수 있는 검색 기능입니다.

- 제목, 저자, 발행연도, 학술지명 등 다양한 기준으로 검색
- 키워드 기반 통합 검색
- 연도별 필터링
- 검색 결과의 정렬 기능 (최신순, 제목순 등)

### 4. 반응형 UI

모바일과 PC 환경 모두에 최적화된 사용자 인터페이스입니다.

- Tailwind CSS 기반의 유틸리티 퍼스트 스타일링
- 모바일 우선 반응형 레이아웃
- 직관적인 논문 카드/리스트 뷰
- 깔끔하고 학술적인 디자인

## 사용 시나리오

### 시나리오 1: 논문 등록

연구원이 새로 발표한 논문의 정보를 시스템에 등록합니다.

1. 메인 페이지에서 "논문 등록" 버튼을 클릭
2. 논문 등록 폼에서 제목, 저자, 발행연도, 학술지명, 원문 링크, 초록 등을 입력
3. "등록" 버튼을 클릭하여 논문 정보를 저장
4. 메인 페이지의 논문 목록에 새 논문이 표시됨

### 시나리오 2: 논문 검색

외부 방문자가 특정 주제의 논문을 찾습니다.

1. 메인 페이지의 검색창에 키워드를 입력
2. 제목, 저자, 학술지명 등에서 일치하는 논문 목록이 표시됨
3. 연도 필터를 적용하여 결과를 좁힘
4. 관심 논문을 클릭하여 상세 정보 확인
5. 원문 링크를 통해 논문 원문에 접근

### 시나리오 3: 논문 정보 수정

논문 메타데이터에 오류가 있어 수정이 필요합니다.

1. 메인 페이지에서 수정할 논문을 클릭하여 상세 페이지로 이동
2. "수정" 버튼을 클릭
3. 관리자 비밀번호를 입력하여 인증
4. 논문 정보를 수정하고 "저장" 버튼을 클릭
5. 수정된 정보가 반영됨

### 시나리오 4: 논문 삭제

더 이상 유효하지 않은 논문 정보를 삭제합니다.

1. 삭제할 논문의 상세 페이지로 이동
2. "삭제" 버튼을 클릭
3. 관리자 비밀번호를 입력하여 인증
4. 삭제 확인 대화상자에서 최종 확인
5. 논문이 목록에서 제거됨

## 주요 사용자 흐름

### 메인 흐름: 논문 목록 조회

```
메인 페이지 접속 -> 논문 목록 표시 -> 검색/필터 적용 -> 논문 선택 -> 상세 정보 조회
```

### 등록 흐름

```
메인 페이지 -> "논문 등록" 클릭 -> 등록 폼 작성 -> 제출 -> 메인 페이지 (목록에 반영)
```

### 수정 흐름

```
논문 상세 페이지 -> "수정" 클릭 -> 관리자 비밀번호 입력 -> 폼 수정 -> 저장 -> 상세 페이지 (변경 반영)
```

### 삭제 흐름

```
논문 상세 페이지 -> "삭제" 클릭 -> 관리자 비밀번호 입력 -> 삭제 확인 -> 메인 페이지 (목록에서 제거)
```

## 논문 메타데이터 구조

각 논문은 다음 정보를 포함합니다:

- **제목** (title): 논문의 전체 제목 (필수)
- **저자** (authors): 저자 목록 (필수)
- **발행연도** (year): 논문이 발행된 연도 (필수)
- **학술지/학회명** (journal): 게재된 학술지 또는 발표된 학회명 (선택)
- **원문 링크** (link): 논문 원문에 접근할 수 있는 URL (선택)
- **초록** (abstract): 논문의 요약 (선택)
- **등록일** (createdAt): 시스템에 등록된 일시 (자동 생성)
- **수정일** (updatedAt): 마지막 수정 일시 (자동 생성)
